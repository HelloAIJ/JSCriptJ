라디오 버튼과 체크 박스에 접근하기(수강신청폼)

라디오 버튼에 접근하기/체크박스에 접근하기/checked 속성을 사용해 선택한 항목에 접근하기

(콘솔도구로 확인하기)

수강신청

[라디오 버튼에 접근하기] 라디오 버튼은 여러 항목이 있어도 하나만 선택된다. 그러나 모든 항목은 결국 '신청 과목'이라는 하나의 덩어리로 name 속성을 사용해서 접근한다. 콘솔 창에서 document.testForm.subject를 입력하면 같은 name값을 가진 요소를 가진 라디오 버튼들은 RadioNodeList라는 노드 리스트 형태로 저장된다. RadioNodeList 앞에 ▶ 를 클릭하면 내부 값을 확인할 수 있다. 아직 선택한 값이 없을 때는 value 값이 ""로 비어있지만 버튼을 선택하고 다시 콘솔에 document.testForm.subject를 입력 해주면 선택한 값의 value 값(ex_문법:grammar)이 나온다.

신청 과목

이 달에 신청할 과목을 선택하세요.

[체크 박스에 접근하기] 체크 박스는 라디오 버튼과는 달리 여러개의 항목을 선택하는 것이 가능하지만

라디오 버튼 처럼 '메일링'이라는 하나의 주제로 묶을 수 있다. 따라서 라디오 버튼 처럼 각 항목의 name을 똑같이 쓰고 name 속성을 사용해서 접근한다.

예를 들어 체크박스에 접근 하려면 콘솔 창에서 document.testForm.mailing을 입력한다. 같은 name값을 가진 check 박스도 RadioNodeList라는 노드 리스트 형태로 저장된다. RadioNodeList 앞에 ▶ 를 클릭하면 내부 값을 확인할 수 있다.

단, 첫번째 박스의 내용을 가져오려면 콘솔에서 document.testForm.mailing[0].value를 입력해야 한다.

메일로 받고 싶은 뉴스 주제를 선택해 주세요

[checked속성을 사용해 선택한 항목에 접근하기] 라디오 버튼이나 체크 박스에는 checked 속성이 있는데 checked 속성은 선택하거나, 선택하지 않는 2개의 값만 가지므로 논리형 값(True, False)으로만 표현된다.

특정 항목을 선택할 경우 그 항목의 속성은 true가 되며 특정 항목을 선택하지 않을 경우에는 false가 된다.

그래서 사용자가 어떤 항목을 선택했는지 알고 싶을 땐 checked 속성을 찾아야 한다. 예를 들어, 라디오 버튼에서 사용자가 '신청과목'을 '문법(name="subject" value="grammar")'으로 선택 시 해당 input 태그에는 checked 속성이 추가된다.

콘솔에 아래와 같이 querySelector 구문을 입력하면 document.querySelector("input[name='subject']:checked")

다음과 같이 확인할 수 있다. input type="radio" name="subject" value="speaking"

단, 체크박스는 항목의 중복 선택이 가능하므로 항목 선택 후 querySelectorAll 구문으로 값을 확인한다.

해외단신(name="new"), 5분회화(name="pops") 선택 document.querySelectorAll("input[name='mailing']:checked") ▶ NodeList(2)[input#new, input#pops]

위와 같이 노드 리스트 형태로 선택한 항목들이 저장되어 보여진다.
