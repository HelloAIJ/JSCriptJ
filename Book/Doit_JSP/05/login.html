DOM의 기초 : DOM에서 폼 다루기

폼 요소에 접근하고 값을 가져오는 방법(1분복습)

폼 요소에 id나 class, name 속성까지 없으면 폼 배열을 사용해 폼 요소에 접근할 수 있음

document의 속성 중 forms 속성은 form 태그를 배열처럼 저장하는데 이는 문서에 form이 두개 이상일 수 있기 때문이다.

또한, form안에 모든 요소를 가져오려면, elements 속성과 함께 사용한다.

폼 배열을 사용해 폼 요소에 접근하기(회원가입폼)

(콘솔도구로 확인하기)

웹 개발자 도구 > [요소(Elements)]탭 클릭 후, 태그 확인한 후 콘솔창에 document.forms를 입력하면 ▼ HTML Collection이라고 표시됨 ▶ HTMLCollection을 클릭하면 1개의 form 요소가 있으며, 아랫쪽에 ▶ 를 계속해서 따라가면 안쪽에 저장된 배열을 확인 할 수 있다. 폼에 접근한 후 그 안에 있는 요소들은 elements 속성을 사용한다. document.forms[0].elements 해당 결과값에서 id 필드에 접근하려면, id 필드가 첫번째([0]) 요소이므로 document.forms[0].elements[0] 이런 식으로 폼 요소를 사용해 id 필드에 접근할 수 있다. document.forms[0].elements[0].value 입력한 값을 콘솔에서 확인 할 수 있다.
