선택 목록에 접근하기(수강신청하기폼)

(콘솔도구로 확인하기)

웹 개발자 도구 > [요소(Elements)]탭을 클릭하면 태그를 확인할 수 있다. select 태그에 id 값이 있으므로 querySelector()함수를 사용해 접근 가능하다. document.querySelector("#major") 선택 목록에 옵션 항목을 접근하려면 options 프로퍼티를 사용한다. document.querySelector("#major").options HTMLOptions Collection(7)이 나타나고 앞쪽에 ▶ 를 클릭해 계속 따라가면 안쪽에 저장된 배열을 확인할 수 있다. length 속성에는 옵션 항목에 갯수가, selectedIndex에는 사용자의 선택이 저장된다. 옵션 항목 중 '산업공학과'를 선택한 후 다시 document.querySelector("#major").options로 확인하면 selectedIndex값이 3으로 변경되었음을 확인할 수 있다.
