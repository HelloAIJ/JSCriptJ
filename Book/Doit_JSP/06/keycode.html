키보드 이벤트

키보드 이벤트의 종류 keydown : 키를 누르는 동안 keypress : 키를 눌렀을 때 keyup : 키에서 손을 뗄 때

키보드에서 키를 눌렀을 때 어떤 키인지 나타내는 이벤트 구현하기

(웹 개발자 도구 : F12)

웹 개발자 도구 > 소스[(Sources)]탭을 클릭하면 해당 js 파일을 확인할 수 있다. const로 body와 result를 선언하고 result에 나타내어질 이벤트 내용을 기입한다. body.addEventListener("keydown",(e)=>{조건}) 조건 : result.innerText=`code : ${e.code}, key : ${e.key}`;
