마우스 이벤트

마우스 이벤트의 종류 click : HTML 요소를 클릭 했을 때 dblclick : HTML 요소를 더블클릭 했을 때 mousedown : 요소에서 마우스 버튼을 눌렀을 때 mousemove : 요소에서 마우스 포인터를 움직일 때 mouseover : 포인터를 요소위로 옮길 때 mouseout : 포인터가 요소를 벗어날 때 mouseup : 요소 위에 올려 놓은 마우스 버튼에서 손을 땔 때

버튼을 클릭하면 배경색이 변경되는 이벤트구현하기

(웹 개발자 도구 : F12)

웹 개발자 도구 > 소스[(Sources)]탭을 클릭하면 해당 js 파일을 확인할 수 있다. document.querySelector("")를 사용하여 이벤트가 실행될 button의 위치를 지정하고 이를 const로 선언한다. button.onclick으로 배경을 변경하는 style 이벤트를 추가한다. 하단의 버튼을 클릭하여 자바스크립트의 동작을 보자.

위의 버튼은 1회성으로 변화된다. 아래의 버튼은 토글 기능을 적용한 버튼이다.
