이벤트와 이벤트 처리기 : 이벤트 알아보기

문서 로딩 이벤트(alert로 확인하기)

문서 로딩 이벤트의 종류 abort : 웹 문서가 완전히 로딩되기 전에 불러오기를 멈추었을 때 error : 문서가 정확히 로딩되지 않았을 때 load : 문서로딩이 끝났을 때 resize : 문서 화면의 크기가 바뀌었을 때 scroll : 문서 화면이 스크롤되었을 때 unload : 문서를 벗어날 때

load 이벤트 구현하기 : window.onload = alert("안녕하세요?");

(웹 개발자 도구 : F12)

웹 개발자 도구 > 소스[(Sources)]탭을 클릭하면 해당 js 파일을 확인할 수 있다. window.onload를 사용하여 알림(alert) "안녕하세요?"를 출력하도록 되어 있는데. 그 외에도 onload에서 on+이벤트명으로 다양한 알림을 줄 수 있다. 단, 각각의 이벤트를 정상적으로 사용하기 위해서는 조건을 별도 지정해야 한다.

창의 높이를 조정하거나 스크롤을 해보세요.
